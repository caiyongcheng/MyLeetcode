 示例 1： 输入：satisfaction = new int []{-1,-8,0,5,-9} 输出：14 解释：去掉第二道和最后一道菜，最大的 like-time 系数和为 (-1*1 + 0*2 + 5*3 = 14) 。每道菜都需要花费 1 单位时间完成。 示例 2： 输入：satisfaction = new int []{4,3,2} 输出：20 解释：可以按照任意顺序做菜 (2*1 + 3*2 + 4*3 = 20) 示例 3： 输入：satisfaction = new int []{-1,-4,-5} 输出：0 解释：大家都不喜欢这些菜，所以不做任何菜就可以获得最大的 like-time 系数。
